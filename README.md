## Panduan Instalasi

1. Clone Repository

git clone https://github.com/kayyerl-nath/Kelompok-Bebas-SNEIK.git
cd Kelompok-Bebas-SNEIK

2. Instal library bila diperlukan

Jika game menggunakan library tambahan (misal curses, pygame, dll.), instal dengan:

pip install -r requirements.txt
(Jika tidak ada requirements, bagian ini dapat dilewati.)

## Panduan Menjalankan Program

Jalankan file utama:

python Bebas.py

atau, jika nama file berbeda, sesuaikan dengan:

python <nama_file>.py

Setelah dijalankan, ikuti instruksi yang muncul pada terminal untuk memainkan game.

## Penjelasan Singkat

1. Program mulai dengan membuat ular, makanan, skor awal, dan arena permainan.


2. Pada setiap frame: pemain memberikan input arah.


3. Ular bergerak dan posisinya diperbarui.


4. Program memeriksa apakah ular:

memakan makanan → skor + panjang bertambah

menabrak dinding atau dirinya sendiri → game over


5. Loop berlangsung hingga kondisi game over terpenuhi.
